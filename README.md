# My Way

Reusable operating guidance for AI-assisted projects: principles, archetypes, tool catalogs, skills, and templates that can be linked from any project.

## How To Use

1. Pick the closest archetype in `archetypes/`.
2. Copy or adapt the matching `templates/agents-md/<archetype>.md` into the target project as `AGENTS.md`.
3. Add the matching `templates/claude-md/<archetype>.md` only when Claude Code needs a `CLAUDE.md` shim.
4. Link to relevant principles, tools, MCP servers, and skills instead of copying long guidance into every project.

This repository is documentation-first. It intentionally does not ship a bootstrap CLI yet.
