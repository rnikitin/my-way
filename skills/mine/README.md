# My Skills

Original or personally maintained skills authored by Roman Nikitin.

## Structure

Each subdirectory is one canonical skill with a `SKILL.md` file and any supporting references needed by that workflow. These skills may be copied into local agent environments or referenced from project instructions.

## How To Navigate

Use this directory when the project needs a workflow that is maintained here rather than copied from an external source. For copied public workflows, use [../_vendored/](../_vendored/). For source notes only, use [../_refs/](../_refs/).

| Skill | Purpose |
|-------|---------|
| [code-simplifier](code-simplifier/SKILL.md) | Simplify code for clarity while preserving behavior and keeping diffs tight. |
| [execplan](execplan/SKILL.md) | Create and execute self-contained plans for long-running, resumable implementation work. |
| [frontend-design](frontend-design/SKILL.md) | Design and implement distinctive production-grade frontend interfaces. |
