# Official MCP Registry

The official MCP registry is the authoritative catalog for public MCP servers. `my-way/mcp/` should not become a competing registry.

Use `my-way/mcp/` for curated archetype guidance: when a server is mandatory, recommended, or optional; what secrets it needs; and when it is overkill. Each MCP entry should include `registry_url` when listed or `registry_status: not-found-YYYY-MM-DD` when not found.

Refresh MCP entries periodically because registry coverage and server quality change quickly.
