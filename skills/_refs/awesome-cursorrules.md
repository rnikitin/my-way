# Awesome Cursor Rules

This community collection is useful for examples and prior art, but it is not the canonical source for Cursor's current rules format.

Use it to inspect how teams phrase stack-specific rules. Do not copy rules wholesale; most examples include assumptions that will be wrong for a different project.

When writing `my-way` Cursor adapters, use the official Cursor docs first and this repository only as inspiration.
