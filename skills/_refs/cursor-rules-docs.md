# Cursor Rules Docs

Cursor's current rules system uses project rules under `.cursor/rules/*.mdc`, with metadata plus Markdown content. Legacy `.cursorrules` should not be the target for new templates.

For `my-way`, Cursor support should be an adapter generated from the canonical `AGENTS.md` content where possible. Do not maintain a separate hand-written rule set unless a project uses Cursor heavily.

This source is canonical for Cursor-specific rule shape.
