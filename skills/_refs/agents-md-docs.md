# Codex AGENTS.md Docs

OpenAI's Codex docs define `AGENTS.md` as the repo-local instruction file for coding agents. It is the right place for project-specific commands, test instructions, style rules, boundaries, and navigation pointers.

For `my-way`, this source makes `AGENTS.md` the canonical shared instruction file. Other tool-specific files should import, point to, or derive from it instead of duplicating content.

Use this reference when creating or reviewing `templates/agents-md/*`.
