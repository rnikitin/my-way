# Simon Willison Agentic Engineering Patterns

Simon Willison frames agentic engineering as a pattern catalog. The useful recurring ideas for `my-way` are red/green TDD, templates, and hoarding useful context into reusable artifacts.

The local takeaway: code generation is cheap, so the scarce resources are intent, taste, validation, and memory. Templates and durable notes help preserve those scarce resources across sessions.

Use this as a living pattern reference rather than a strict process document.
