# AGENTS.md Open Spec

The AGENTS.md project is a lightweight open convention for storing agent instructions in a repository. Its examples focus on development environment tips, testing instructions, and pull request expectations.

For `my-way`, this is the cross-tool anchor. Codex reads `AGENTS.md` directly; Claude Code gets the same content through a thin `CLAUDE.md` import shim; Cursor or other tools can adapt from the same source.

Do not treat this as permission to create long root files. Pair it with instruction minimalism.
