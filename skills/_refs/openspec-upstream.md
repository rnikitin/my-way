# OpenSpec Upstream

OpenSpec is a lightweight spec-driven development framework for AI coding assistants. It creates change folders with proposal, design, specs, and tasks so humans and agents align before implementation.

The workflow is useful when a request adds capability, changes architecture, touches public contracts, or is ambiguous enough that chat-only intent will drift. It is deliberately lighter than heavyweight formal planning and supports many agent tools through slash commands and skills.

For `my-way`, OpenSpec is not the default for every task. It belongs in `agent-workflow`, data-platform, and product-heavy archetypes when changes need reviewable requirements and archival closeout.
