# AGENTS.md Templates

This directory contains hand-curated `AGENTS.md` templates by project archetype. These files are meant to be copied into a target repository, edited down, and treated as the canonical local instructions for Codex and other AGENTS.md-aware agents.

## Structure

Each template should stay compact and use the same navigation shape:

- commands
- testing
- project structure
- code style
- git workflow
- boundaries: always do, ask first, never do

## How To Navigate

Choose a template from the [archetype](../../archetypes/) that matches the target project. After copying it, replace placeholders with real local commands and remove generic text. Put long guidance in linked docs or skills, not in the root `AGENTS.md`.
